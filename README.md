# YRBS Sleep & Social Media Analysis

## 📊 Project Overview

This project analyzes the relationship between **sleep duration** and **social media usage** among adolescents using data from the Youth Risk Behavior Survey (YRBS).

The goal is to explore patterns, identify trends, and support findings with statistical analysis and data visualization.

---

## ❓ Research Question

Is there a relationship between social media usage and sleep duration among students?

---

## 📁 Dataset

The dataset used in this project is from the **2023 National Youth Risk Behavior Survey (YRBS)**.

---

## 🔢 Variables

The analysis focuses on the following variables:

* **Q90** – Social media usage
* **Q96** – Hours of sleep on an average school night

These variables were selected to explore the relationship between social media use and sleep patterns among students.

---

## ⚙️ Methods

The project includes:

* Data cleaning and preprocessing
* Grouping of variables (e.g., social media usage levels)
* Handling missing values using `dropna`
* Exploratory Data Analysis (EDA)
* Statistical analysis:

  * Mean, median, standard deviation
  * Correlation analysis
  * Hypothesis testing (p-value)

---

## 📈 Visualizations

We used data visualization to better understand patterns:

* Bar charts
* Boxplots / stripplots
* Distribution analysis

These visualizations help explain the relationship between variables.

---

## 📊 Key Findings

* The results suggest a relationship between social media usage and sleep duration
* Higher social media usage tends to be associated with lower sleep duration
* Statistical analysis supports the observed patterns
